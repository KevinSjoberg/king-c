# Exercise

Show the output produced by each of the following program fragments. Assume
`i`, `j` and `k` are `int` variables.

```c
i = 5, j = 3;
printf("%d %d\n", i / j, i % j);
```

```c
i = 2, j = 3;
printf("%d\n", (i + 10) % j);
```

```c
i = 7, j = 8, k = 9;
printf("%d\n", (i + 10) % k / j);
```

```c
i = 1, j = 2, k = 3;
printf("%d\n", (i + 5) % (j + 2) / k);
```
